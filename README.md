# Google Keep Clone

## Overview

The Google Keep Clone is a responsive web application developed using **HTML5**, **CSS3**, and **JavaScript**. The application recreates the core functionality of Google Keep, allowing users to create, edit, archive, delete, restore, and search notes through a clean and intuitive interface.

## Features

- Create, edit, and delete notes
- Archive and restore notes
- Search notes by title or content
- Toggle between Grid and List view
- Collapsible sidebar navigation
- Responsive design for different screen sizes
- Interactive edit modal

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap Icons

## Getting Started

1. Download or clone the repository.
2. Open the project in Visual Studio Code.
3. Launch `index.html` using Live Server or open it in your preferred web browser.
